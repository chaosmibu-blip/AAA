# Chrome Web Store 上架資訊

## 擴充名稱
指令小幫手 - Command Helper

## 簡短描述（132 字元以內）
快速複製常用指令的懸浮球工具，內建提問模板幫你問 AI 好問題

## 詳細描述

🎯 **指令小幫手** 是專為開發者和 AI 使用者設計的效率工具。

### ✨ 主要功能

📋 **快速複製指令**
- 一鍵複製常用的 Git、開發指令
- 懸浮球設計，隨時可用不打擾工作
- 指令分類管理，快速找到需要的內容

💡 **提問模板**
- 內建多種高品質提問模板
- 幫你問 AI 更精準的問題
- 包含程式碼審查、錯誤解釋、重構建議等模板

🔧 **自訂指令**
- 新增自己常用的指令
- 支援收藏功能，常用指令優先顯示
- 本地儲存，資料不會遺失

### 📦 內建模板分類

- **提問模板**：請求程式碼審查、解釋錯誤訊息、寫單元測試、重構建議
- **Git 指令**：commit、push、pull、branch 等常用指令
- **開發指令**：npm run dev、npm install、npm test 等

### 🎨 使用方式

1. 點擊網頁右下角的懸浮球
2. 選擇分類標籤
3. 點擊指令即可複製
4. 貼到 ChatGPT、Claude 或終端機使用

### 🔒 隱私保護

- 所有資料存在本地，不會上傳到任何伺服器
- 不收集任何使用者資訊
- 開源透明

## 上架步驟

1. 前往 https://chrome.google.com/webstore/devconsole
2. 註冊開發者帳號（一次性費用 $5 美元）
3. 點擊「新增項目」
4. 上傳 extension 資料夾的 ZIP 檔
5. 填寫上方資訊
6. 上傳截圖和圖標
7. 提交審核（約 1-3 天）
